## Conference Ticket Generator UI

This is a project that aims to reflect a Ticket Generator.

### Built With:

- Next.js,
- TypeScript,
- Tailwind CSS,
- daisyUI.

### Used Methods/Hooks:

- useState,
- useEffect,
- map(),
- Conditional Rendering (&& and ?),
- Array destructuring,
- Regexes.

Check it out on [here](https://conference-ticket-gen-brown.vercel.app) !
